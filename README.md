💊 Online Pharmacy Portal

A web-based application designed to streamline pharmacy operations including medicine browsing, user registration, and order management.

📌 Project Overview

This Online Pharmacy Portal was developed as a Group Project for the 1st Year, 2nd Semester.
The system allows users to browse available medicines, create accounts, and place orders, while enabling the pharmacy to manage medicines and customer requests efficiently.

🚀 Features

👤 User Features

User registration & login

Browse available medicines

Add medicines to cart

Place orders

🏥 Admin Features

Manage medicine catalog (Add / Edit / Delete)

View & manage customer orders

Update stock availability

🛠️ Technologies Used

HTML – Frontend structure

CSS – Styling and layout

JavaScript – Interactions & validations

PHP – Server-side logic

MySQL / SQL – Database management
